# Angiospermas APG IV

Infografía web interactiva para presentar familias representativas de angiospermas según APG IV.

## Ejecutar localmente

```bash
npm install
npm run dev
```

## Deploy en Vercel

1. Entrar a Vercel.
2. Importar este repositorio de GitHub.
3. Framework: Vite.
4. Build command: `npm run build`.
5. Output directory: `dist`.
